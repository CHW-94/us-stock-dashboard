# 📊 미국 주식 분석 대시보드

실시간 미국 주식 데이터를 분석하는 웹 대시보드입니다.

## 🎯 주요 기능

- 실시간 주식 가격 조회
- 역추세 매매 전략 (과매도 매수) 시그널 분석
- RSI, 볼린저밴드, MACD, 이동평균선 등 기술적 지표
- 30일 가격 차트
- 프리마켓/애프터마켓 데이터

## 🚀 사용 방법

1. 티커 심볼 입력 (예: AAPL, TSLA, NVDA)
2. 검색 버튼 클릭 또는 인기 종목 버튼 클릭
3. 분석 결과 확인

## 🔧 기술 스택

- HTML5 + JavaScript (Vanilla)
- Yahoo Finance API
- CORS 프록시 서버 (4개 자동 전환)

## ⚠️ 주의사항

이 분석은 기술적 지표에 기반한 참고용입니다.
투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.
